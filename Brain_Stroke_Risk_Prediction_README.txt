Brain Stroke Risk Prediction ML

This repository contains a machine learning based web application
that predicts the risk of brain stroke using patient data.

Features:
- Data preprocessing and imbalance handling
- Model comparison (Logistic Regression, SVM, Random Forest)
- Threshold tuning for recall optimization
- FastAPI backend with predictive API
- Interactive frontend

How to Run:
1) Install dependencies:
   pip install -r requirements.txt

2) Run server:
   uvicorn app:app --reload

Visit in browser: http://127.0.0.1:8000
